# Kyocera printers by SNMPv2

## Overview

Template created for Kyocera ECOSYS m3655idn.
Works in any Kyocera printer.


Contains items for:

* Printer name
* Cartridge name
* Serial Number
* Printer status
* Total pages printed
* Printed with toner
* Toner capacity
* Toner level (%)


Triggers for:

* Toner < 5%

## Macros used

* {$COMM} for Community Name
